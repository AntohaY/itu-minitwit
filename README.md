## How to run env locally
### 1. Create an .env file in root folder and setup environment variables
    DOCKER_USERNAME=****
    GRAFANA_ADMIN_USER=****
    GRAFANA_ADMIN_PASSWORD=****
### 2. Run this command to start docker
    docker compose up --build
